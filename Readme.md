# AQI Analysis and Prediction Project

## Overview
This project analyzes air quality trends across Indian states and builds machine learning models to predict AQI and classify air quality status.

## Steps Performed
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Model Building (Regression & Classification)
- Model Evaluation

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
